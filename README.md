## 🚀 Kurulum ve Kullanım / Installation & Usage

<img src="https://cdnjs.cloudflare.com/ajax/libs/twemoji/14.0.2/72x72/1f1f9-1f1f7.png" width="16"/> Bu yapılandırma dosyalarını kendi sisteminize uygulamak için ilgili klasörleri `~/.config/` dizininize taşıyabilir veya sembolik bağ (symlink) oluşturabilirsiniz.
<br>
<img src="https://cdnjs.cloudflare.com/ajax/libs/twemoji/14.0.2/72x72/1f1ec-1f1e7.png" width="16"/> To apply these configurations to your system, you can copy the respective directories into your `~/.config/` path or manage them via symlinks.

```bash
git clone [https://github.com/yesnt6969/yesnt69-config-files.git](https://github.com/yesnt6969/yesnt69-config-files.git)
cd yesnt69-config-files

# TR: Örnek - Neovim konfigürasyonunu kopyalamak için
# EN: Example - Copying the Neovim configuration
cp -r nvim ~/.config/

# TR: Önerilen - Değişiklikleri anlık takip edebilmek için sembolik bağ oluşturma
# EN: Recommended - Creating a symlink to track updates easily
ln -s $(pwd)/nvim ~/.config/nvim
